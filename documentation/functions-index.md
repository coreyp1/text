# Function Index

## Overview

This page provides quick access to all functions in the Ghoti.io Text library,
organized alphabetically.

## Direct Access

[Complete Function Index](globals_func.html) - Browse all functions alphabetically

## Navigation

You can also access the function index through the navigation menu:
- **Files** → **Globals** → **Functions**

## Search

Use the search box at the top of any page to quickly find specific functions
by name.

> **Note:** The function index is automatically generated from documented functions
> in the library headers. All public API functions are included.
